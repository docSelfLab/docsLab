2. Description of the App

CalmSprint provides productivity tools intended to help you manage focus sessions, including:

  • Focus / Short Break / Long Break timer cycles
  • Task creation, editing, tagging, completion, and binding tasks to a focus session
  • Local statistics such as daily goals, streaks, weekly summaries, and task rankings
  • Optional local notifications for phase completion and monthly reminders
  • Home Screen widgets and Live Activities / Dynamic Island countdown (where supported)
  • Appearance options (theme, accent color) and language preferences (System / Chinese / English)

The App is designed to work offline. Core features do not require an account or internet connection.
